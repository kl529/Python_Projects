# Python_matgo

### matgo with python
파이썬을 이용한 맞고 게임입니다. Python-GUI를 통하여 상호작용을 클릭으로 할 수 있으며, 난이도 별 AI 단계 적용이 되어있습니다.

## 사용방법
1. clone the project
2. open Final Folder
3. command "python matgo.py"
4. Enjoy!!
